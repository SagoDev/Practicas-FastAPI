# Plan de Ejercicios Prácticos – FastAPI

Este repositorio contiene un **planes de ejercicios prácticos** diseñado para reforzar el uso y conocimiento de **FastAPI**.

El enfoque es **progresivo**, orientado a la práctica y alineado con buenas prácticas de diseño de APIs REST.

---

## Objetivo

El objetivo principal es que el desarrollador:

* Comprenda cómo FastAPI maneja parámetros de ruta y query
* Aplique tipado y validación automática
* Diseñe endpoints REST claros y documentados
* Utilice Swagger/OpenAPI como herramienta de validación

---

## Recomendaciones de Uso

* Resolver los ejercicios **en orden**
* Probar cada endpoint desde Swagger UI
* Forzar errores para analizar respuestas `422`
* Comparar path parameters vs query parameters

---

## Buenas Prácticas

* Tipar siempre los parámetros
* Usar `Query()` para validaciones y documentación
* Mantener endpoints claros y coherentes
* Respetar principios REST
